👇只需准备一次机器人 ---> 10分钟👇

1. 将机器人添加到您的频道。

2. 将机器人设置为频道管理员。 💥机器人必须具备发送消息的权限💥

3. 您需要获取聊天的“ID”，例如通过 @getidsbot。  
   将您计划发布内容的频道中的任意一条消息转发给机器人 - 💥这是免费的💥

4. 您将收到类似如下的消息：

👤 You
 ├ id: 123456789
 ├ is_bot: false
 ├ first_name: Mr
 ├ last_name: Smith
 ├ username: e_x_a_m_p_l_e (https://t.me/example)
 ├ language_code: en (-)
 └ created: ~ 12/2017 (?) (https://t.me/getidsbot?start=idhelp)

💬 Origin chat
 ├ id: -1000101010101  <----- 💥您需要这个“ID”💥
 ├ title: test_public
 ├ username: e_x_a_m_p_l_e (https://t.me/example)
 └ type: channel

📃 Message
 ├ message_id: 2 (https://t.me/dfahasdj/2)
 └ forward_date: Thu, 09 Apr 2026 23:38:26 GMT

5. 打开 "NEO_TEMPLATE.xls"  
6. 填写字段

channel:  
   --> 必填字段  
   --> 频道聊天ID（第4步）  
   示例: -1000101010101

title:  
   --> 必填字段  
   --> AI使用它在互联网上搜索信息（不超过200字符）  
   --> 这是标题（显示给频道成员）  
   示例: "非洲旅游"

keywords:  
   --> 必填字段  
   --> AI使用它在互联网上搜索信息（不超过200字符）  
   --> 这些是程序用于查找资料的标签  
   示例: "非洲，旅游，大象，酒店，野生动物园，专家风格"

response_language:  
   --> 可选字段。默认 "ENG"  
   --> 文章语言。仅支持1种语言。  
   --> en - 英语，es - 西班牙语，uk - 乌克兰语，zh - 中文，hi - 印地语，ar - 阿拉伯语，ru - 俄语  
   示例: en

include_image:  
   --> 必填字段  
   --> 是否在文章中添加图片  
   --> True - 带图片，False - 不带图片  
   --> 图片根据标题和文章主题自动生成

footer_text:  
   --> 可选字段  
   --> 文章末尾的文本（不超过200字符）  
   示例: 感谢您的支持！！在TikTok找到我们！！

footer_link:  
   --> 可选字段  
   --> 文章末尾的链接（不超过200字符）  
   示例: https://t.me/+exuWbQJ8pBFhMzEy 或 https://www.youtube.com/@JavaRangers

schedule_at:  
   --> 必填字段  
   --> 指定发布时间  
   --> 如果时间已经过去 --> 机器人会返回错误！  
   --> 格式 YYYY-MM-DD HH:MM  
   --> 中欧时间（马德里）。请在设置时间时注意  
   示例: 2026-12-31 23:59


🎯 重要！如果以文本形式输入：仅支持 YYYY-MM-DD HH:MM 格式，例如: 2026-04-13 18:25  
          最佳方式：使用Excel的Date/Time类型单元格（非文本）——任意格式均可

mode:  
   --> 必填字段  
   --> 工作模式：  
       --> instant - 在指定时间自动发布到频道  
           示例: 2026-12-31 23:59 - 表示将在该时间发布（可能延迟最多1-2分钟）

       --> approval - 您将在机器人中收到文章已准备好的通知。之后可以选择  
           --> 按钮 [发布]- 立即发布到频道  
           --> 按钮 [下载归档] - 您将收到包含所有文章的ZIP压缩包。点击 [下载归档] 后可以  
               自行检查文章，并上传到您的频道（可编辑或保持原样）。  
               💡 **在浏览器中打开 HTML 文件以获得更好的可读性**
           点击 [下载归档] 后任务视为完成。  
           示例: 31:12:2026 23:59 - 此时机器人将提示发布或下载ZIP

💥 重要！如果在 ~10 分钟内未做出决定 —  
   机器人将自动发送包含所有任务内容的ZIP压缩包（不发布）

💡 为什么需要这样：  
   — 防止未经确认自动发布内容  
   — 即使您忙碌也能获得结果  
   — 方便您后续检查和使用内容        

7. 填写完成后，将 "NEO_TEMPLATE.xls" 发送给机器人🎯  
8. 机器人将检查数据  
   如果有错误 --> 会提示错误位置  
   如果没有错误 --> 会通知已开始处理
