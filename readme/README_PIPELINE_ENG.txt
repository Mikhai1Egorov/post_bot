 channel: 
    --> REQUIRED FIELD
    --> Channel chat ID (p.4)
    EXAMPLE: -1000101010101

 title:
    --> REQUIRED FIELD
    --> AI uses it to search for information on the internet ( no more than 200 characters )

 keywords:
    --> REQUIRED FIELD
    --> AI uses it to search for information on the internet ( no more than 200 characters )

    --> REQUIRED FIELD   
    --> Add an image to the post or not

 footer_text:
    --> OPTIONAL FIELD
    --> Your text at the end of the article ( no more than 200 characters )
    EXAMPLE: Thank you for being with us!! Find us on TikTok!!

    --> OPTIONAL FIELD
    --> Your link to any resource at the end of the article ( no more than 200 characters )
    EXAMPLE: https://t.me/+exuWbQJ8pBFhMzEy OR https://www.youtube.com/@JavaRangers

    --> If the time has ALREADY PASSED --> the bot will return an error!
    --> Format YYYY-MM-DD HH:MM
    --> CENTRAL EUROPEAN TIME (MADRID). CONSIDER THIS WHEN SETTING TIME
    EXAMPLE: 2026-12-31 23:59


   🎯 IMPORTANT! If entering as TEXT: only format YYYY-MM-DD HH:MM example: 2026-04-13 18:25.
             Best option: Excel cell type Date/Time (not text) - then any format works.
